--- layout: post name: fix-fedora-boot-label-error-message title: Fix Fedora boot label error message time: 2015-04-09 03:27:00.000000000 +00:00 --- In this video I show you how to fix the Fedora boot label error message: "Warning: /dev/disk/by-label/Fedora-Live-WS-x86_64-21-5 does not exist." It may differ depending on the version of Fedora you are running, and this mostly comes up when installing via a USB drive. You need to watch how you label your boot media, and if needed change the label while booting to what ever you named the media (USB/DVD/CD).
