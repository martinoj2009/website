I've had this problem a couple times, it's because I don't have to do it often. Once it's installed I don't have to do it again unless I uninstall it for some reason. If you already have VirtualBox and you just want to update to the new version you can follow this as well, you just don't have to install VirtualBox. After doing this the new version should be in your updates.

Don't use the Ubuntu repository

I would recommend getting VirtualBox from them, not through the Ubuntu repository. It's typically out of date, or it has had the problem when I last used it. For the past couple of years I just get it from the devs. It's not hard to do really, you just need to add their repository and key.

Installation

The process REALLY isn't that hard. Just takes 4 commands:

Fist, we need to add the repository:

{% highlight text %} sudo sh -c 'echo "deb http://download.virtualbox.org/virtualbox/debian $(lsb_release -cs) contrib" >> /etc/apt/sources.list.d/virtualbox.list' {% endhighlight %}

Next, we need to add the GPG key:

{% highlight text %} sudo wget -q https://www.virtualbox.org/download/oracle_vbox.asc -O- | sudo apt-key add - {% endhighlight %}

Now, we need to update the repos:

{% highlight text %} sudo apt-get update {% endhighlight %}

Finally, we need to install VirtualBox:

I will be doing this for VirtualBox 5... download what version you want. {% highlight text %} sudo apt-get install virtualbox-5.0 {% endhighlight %}

Now join the group

Now that we have VirtualBox installed you should add your account to the vboxusers group:

{% highlight text %} sudo usermod -a -G vboxusers $USER {% endhighlight %}

Done

That's it, now you MAY need to reboot your machine, but I didn't have to.
